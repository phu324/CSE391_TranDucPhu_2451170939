⌨️ Hướng dẫn phím tắt bàn phím:

  • Ctrl + K: Bật/Tắt Menu Lệnh (Command Palette)
  • ← →: Chuyển đổi ảnh kế tiếp / lùi lại trong bộ sưu tập
  • 1 - 9: Nhảy nhanh tới vị trí ảnh tương ứng
  • Space (Dấu cách): Bật / Tạm dừng Trình chiếu ảnh (Slideshow)
  • Esc: Đóng nhanh Modal chi tiết hoặc Command Palette
  • Tab: Di chuyển tiêu điểm Focus qua các nút điều khiển

🖼️ Bộ Sưu Tập Ảnh Tương Tác

Bấm Enter để chạy, Esc để thoát
