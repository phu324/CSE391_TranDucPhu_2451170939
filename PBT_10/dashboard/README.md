# 📊 Bảng Điều Khiển Đa Dữ Liệu (Multi-API Dashboard) - Bài B4


## 🌐 APIs Sử dụng
Ứng dụng tích hợp đồng thời 3 hệ thống dữ liệu:
1. **Random User API:** `https://randomuser.me/api/` (Lấy dữ liệu thành viên ngẫu nhiên)
2. **Dog API:** `https://dog.ceo/api/breeds/image/random` (Lấy hình ảnh thú cưng)
3. **CoinGecko API:** `https://api.coingecko.com/` (Tải tỷ giá các đồng tiền mã hóa Bitcoin, Ethereum...)

## 🚀 Hướng dẫn cách chạy ứng dụng
1. Truy cập vào thư mục `dashboard/`.
2. Mở file `index.html` trên trình duyệt.
3. Quan sát thời gian tải ở góc phải trên và nhấn nút **Refresh All** để cập nhật lại thông tin mới.
