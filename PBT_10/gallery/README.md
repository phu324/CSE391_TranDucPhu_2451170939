# 🌌 Bộ Sưu Tập Ảnh Vô Tận (Infinite Scroll Gallery) - Bài B3


## 🌐 API Sử dụng
- **Dịch vụ cung cấp:** `Lorem Picsum`
- **Đường dẫn phân trang:** `https://picsum.photos/v2/list?page=[Số_Trang]&limit=20`


## 🚀 Hướng dẫn cách chạy ứng dụng
1. Truy cập vào thư mục `gallery/`.
2. Khởi chạy file `index.html` trên trình duyệt.
3. Tiến hành cuộn chuột liên tục xuống phía dưới để quan sát hiệu ứng tải thêm ảnh vô hạn.
