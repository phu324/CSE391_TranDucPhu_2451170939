# 👥 Hệ Thống Quản Lý Thành Viên (User Directory CRUD) - Bài B2


## 🌐 API Sử dụng
- **Dịch vụ cung cấp:** `JSONPlaceholder`
- **Các Endpoints xử lý:**
   - `GET    /users`      - Lấy danh sách thành viên ban đầu
   - `POST   /users`      - Tạo tài khoản thành viên mới
   - `PUT    /users/:id`  - Cập nhật thông tin thành viên
   - `DELETE /users/:id`  - Xóa thành viên khỏi hệ thống

## 🚀 Hướng dẫn cách chạy ứng dụng
1. Truy cập vào thư mục `user_directory/`.
2. Mở file `index.html` bằng trình duyệt Web.
3. Sử dụng thanh công cụ để tìm kiếm thành viên, hoặc nhấn **Thêm thành viên** để kích hoạt Form nhập liệu.
