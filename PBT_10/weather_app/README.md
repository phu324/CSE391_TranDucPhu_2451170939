# 🌤️ Ứng Dụng Thời Tiết (Weather App) - Bài B1


## 🌐 API Sử dụng
- **Dịch vụ cung cấp:** `wttr.in`
- **Đường dẫn gọi dữ liệu:** `https://wttr.in/[Tên_Thành_Phố]?format=j1`
- **Định dạng dữ liệu trả về:** JSON


## 🚀 Hướng dẫn cách chạy ứng dụng
1. Truy cập vào thư mục `weather_app/`.
2. Click đúp chuột vào file `index.html` để mở trực tiếp trên trình duyệt Web (Chrome, Edge, Firefox).
3. Nhập tên thành phố bằng tiếng Anh không dấu (Ví dụ: `Hanoi`, `Tokyo`, `London`) rồi nhấn **Tìm** hoặc nhấn **Enter**.
